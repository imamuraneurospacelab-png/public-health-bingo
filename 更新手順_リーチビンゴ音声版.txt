【リーチ・ビンゴ音声版 更新手順】

今回は、ブラウザの自動読み上げとは別に、固定音声ファイルを追加しています。
そのため「リーチ！」「ビンゴ！」は、問題読み上げが動かない環境でも再生できます。

1. GitHubで public-health-bingo を開く
2. public フォルダを開く
3. Add file → Upload files
4. このZIPを解凍し、次のものをアップロード

   app.js
   audio フォルダ（reach.mp3 / bingo.mp3）

5. Commit changes を押す
6. Cloudflareの自動デプロイが完了するまで1〜2分待つ
7. 公開ページで Command + Shift + R を押して再読み込み

【動作】
・学生があと1マスになると、学生スマホと教員画面で「リーチ！」
・ビンゴ成立時は「ビンゴ！」
・従来の効果音、紙吹雪、順位表示も継続
・問題の自動読み上げ処理も、音声が途切れにくいよう調整

【本番前の確認】
教員PC 1台＋スマホ1台でルームを作り、正答を続けてリーチとビンゴ音声を確認してください。
スマホは消音モードを解除し、メディア音量を上げてください。
