教員側「リーチ／ビンゴ」固定音声版 更新手順

1. このZIPを解凍します。
2. GitHubの public-health-bingo リポジトリを開きます。
3. public フォルダを開きます。
4. Add file → Upload files を押します。
5. このZIP内の app.js をドラッグします。
6. 「app.js already exists」と表示されても、そのまま置き換えて Commit changes を押します。
7. Cloudflareの自動デプロイ完了を1〜2分待ちます。
8. 教員画面を Command + Shift + R で強制再読み込みします。
9. 教員画面の「音声を有効にする」を押します。
10. その場で「リーチ」と聞こえれば準備完了です。

変更点
- 音声データをapp.js内に直接埋め込みました。
- audioフォルダの置き場所やブラウザキャッシュに依存しません。
- 各学生の状態変化を教員画面で追跡し、新規リーチ時に「リーチ」、新規ビンゴ時に「ビンゴ」と再生します。
- 音声を有効にするボタンを押した直後に「リーチ」のテスト音声を流します。

注意
- 教員PCの音量とChromeタブのミュートを確認してください。
- 「音声を有効にする」は、授業開始前に必ず一度押してください。
